# FPart

- **Что делает продукт?**
> FPart - русскоязычная Jira. Менеджмент проектов. Управление задачами. Релиз.

- **Кто пользователь?**
> CA - разработчики.

- **Что он может делать в MVP?**
> - Создавать проекты
> - Добавлять участников
> - Удалять проекты
> - Получать информацию:
>    - о пользователе
>    - о проекте
>    - о проектах пользователя
> - Логиниться через Google

- **Что не делает в MVP?**
> - Логиниться как-либо помимо Google
> - Продуманное управление:
>   - задачами
>   - настройками проекта
> - Не собирает данные для анализа


## Стек

- **Frontend**: Vue3 + Vite + TypeScript
- **Backend**: Golang (fasthttp)
- **DB**: Postgres, Redis
- **Infra**: Nginx, Docker
- **Modes**: Dev / Prod
